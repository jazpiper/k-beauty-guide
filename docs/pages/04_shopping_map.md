# Shopping Map 페이지 기획서

> 최종 수정: 2026-04-15

---

## 1. 목적

서울 방문 외국인이 **K-뷰티 매장을 쉽게 찾고** 방문 계획을 세울 수 있도록 돕는 페이지.  
지역·매장 유형 필터로 목적에 맞는 매장을 발견하고,  
영업시간·추천 제품 정보로 방문 전 사전 준비를 지원한다.

## 3. 섹션별 상세 기획

### 3-1. 지도 영역

**현재**
- CSS로 만든 시각적 플레이스홀더
- 핀 위치: `top: 20+(i*11)%65%`, `left: 15+(i*17)%70%` 수식으로 자동 배치 (실제 좌표 아님)
- 핀 클릭 → 매장 상세 패널 표시

**목표 (Phase 2)**
- [ ] **Kakao Maps API** 또는 **Google Maps API** 연동
  - Kakao Maps: 한국 지도 정확도 높음, 한국어 POI 풍부, 무료 티어 있음 (권장)
  - Google Maps: 글로벌 외국인 사용자에게 친숙
- [ ] 매장별 실제 위도/경도 데이터 입력
- [ ] 지도 마커 커스텀 (매장 유형별 색상 구분)
- [ ] 마커 클릭 → 상세 패널 연동

**지도 구현 옵션 비교**

| 옵션 | 장점 | 단점 |
|---|---|---|
| Kakao Maps JS API | 한국 지도 정확, 무료 | 가입 필요, 한국어 문서 |
| Google Maps JS API | 글로벌 친숙, 문서 풍부 | 유료 (월 $200 크레딧) |
| Naver Maps API | 국내 최고 정확도 | 외국인 접근성 낮음 |
| react-leaflet + OSM | 완전 무료, 오픈소스 | 한국 지도 품질 낮음 |

**권장**: Kakao Maps (한국 방문 외국인 대상, 정확도 우선)

### 3-2. 필터

**현재 필터**
- Area: All / Myeongdong / Hongdae / Gangnam / Jung-gu / Dongdaemun
- Type: All / Drugstore / Brand Store / Duty Free / Department Store / Multi-brand / Market

**개선 사항**
- [ ] **현재 영업 중** 토글 필터 (실시간 시간 기반)
- [ ] **지하철역 기준** 필터 (외국인이 이동 계획 세울 때 유용)
  - e.g. "명동역 2번 출구 도보 3분"
- [ ] **예산 필터** — 고가 브랜드(Sulwhasoo) vs 합리적 가격(Olive Young)
- [ ] 필터 결과 수 표시 ("3 stores found")

---

### 3-3. 매장 카드 목록

**현재**: 매장명, 지역, 유형, 영업시간 표시 + 클릭 시 상세 패널

**개선 사항**
- [ ] **지하철 정보** 추가 (역명 + 출구 + 도보 시간)
- [ ] **영업 상태** 표시 — 현재 시간 기준 "영업 중 / 영업 종료" 실시간 표시
- [ ] **거리 정보** — 위치 권한 허용 시 현재 위치에서의 거리
- [ ] **카드 정렬** — 거리순 / 영업시간순 / 인기순
- [ ] **즐겨찾기** 버튼 (LocalStorage 저장)

---

### 3-4. 매장 상세 패널

**현재 표시 항목**
- 이모지 / 매장명 / 지역 / 유형 뱃지
- 설명 / 영업시간
- Must-try 제품 태그 목록

**개선 사항**
- [ ] **지도에서 보기** 버튼 → 지도 해당 마커로 포커스
- [ ] **길찾기** 버튼 → Google Maps / Kakao Maps 앱 딥링크
  - iOS: `kakaomap://route?ep=위도,경도&by=PUBLICTRANSIT`
  - 웹: `https://map.kakao.com/?eName=매장명`
- [ ] **전화번호** 추가 (탭 시 전화 연결)
- [ ] **홈페이지 링크** 추가
- [ ] **Must-try 제품 클릭** → Products 페이지 해당 제품으로 연결
- [ ] **사진** 추가 (매장 외관 / 내부)
- [ ] **혼잡도 정보** (Phase 2: Google Places API)

### 3-5. 지역 가이드 섹션 (신규 기획)

지역별 특성을 설명하는 안내 섹션 추가.

| 지역 | 특징 | 추천 대상 |
|---|---|---|
| Myeongdong (명동) | 최대 K-뷰티 집결지, 외국인 특화, 면세점 밀집 | 모든 방문자 필수 |
| Hongdae (홍대) | 트렌디·힙, 인디 브랜드, 젊은 층 | MZ세대, 트렌드 추구형 |
| Gangnam (강남) | 프리미엄 럭셔리, 성형외과 밀집 | 하이엔드 스킨케어 관심자 |
| Dongdaemun (동대문) | 심야 쇼핑, 도매 가격, 시장 분위기 | 대량 구매·가격 중시형 |
| Jung-gu (중구) | 롯데·신세계 백화점, 면세점 | 브랜드 선물 구매 |

## 4. 데이터 요구사항

### 현재 매장 데이터 구조
```js
{
  name, area, type, emoji, desc, open, must: []
}
```

### 목표 데이터 구조 (Phase 1)
```js
{
  id,
  name,
  area,
  type,
  emoji,
  desc,
  hours: {
    mon: "09:00-23:00",
    tue: "09:00-23:00",
    // ...
    holiday: "휴무 없음"
  },
  must: [],            // 추천 제품명
  product_ids: [],     // Products DB 연결
  lat: 37.5631,
  lng: 126.9822,
  subway: {
    station: "명동역",
    line: "4호선",
    exit: "8번 출구",
    walk_min: 3
  },
  phone: "02-000-0000",
  website: "https://...",
  photos: [],
  is_featured: false
}
```
